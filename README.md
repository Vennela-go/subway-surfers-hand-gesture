# Subway Surfers Hand Gesture

Hand gesture-based control for Subway Surfers using Python, OpenCV, MediaPipe, and PyAutoGUI.

## Features
- Jump, Roll, Move Left, Move Right, Hoverboard using hand gestures
- Finger-based gesture recognition
- Works on PC version, emulator, and browser games (with adjustments)

## Tech Stack
- Python 3.10
- OpenCV
- MediaPipe
- PyAutoGUI
- Numpy
